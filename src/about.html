Cos'è Ayisha.js?

Ayisha.js è un micro-framework JavaScript per la creazione di interfacce web reattive e moderne, pensato per essere semplice, leggero e immediato.
Non richiede build, transpiler o dipendenze esterne: basta includere uno script e scrivere HTML con direttive.

Caratteristiche principali:

  • ⚡ Reattività automatica dello stato
  • ⚡ Direttive per condizioni, cicli, eventi, binding, fetch, validazione e altro
  • ⚡ Routing SPA integrato
  • ⚡ Componenti HTML caricabili anche da file esterni
  • ⚡ Zero build: nessun toolchain, solo HTML + JS
  • ⚡ Ultra-leggerezza: ~20KB minificato
  • ⚡ API ispirata a Vue/Angular ma senza complessità

Ideale per: prototipi, demo, landing page, micro-app, documentazione interattiva, test UI.
Non adatto a: grandi progetti enterprise con esigenze di SSR, TypeScript avanzato, o bundle complessi.
