Info Tecniche su Ayisha.js

  • Dimensione: ~20KB minificato
  • Dipendenze: Nessuna
  • Reattività: Proxy JS, watchers, binding automatico
  • Direttive: @model, @if, @for, @show, @hide, @click, @fetch, @result, @validate, @page, @link, @component, @class, @style, @switch, @case, @default, @state, @log, e molte altre
  • Routing: SPA nativo, gestione URL e history
  • Componenti: HTML inline o caricati da file esterni
  • Validazione: Integrata via direttiva @validate
  • Animazioni: Classi CSS e direttiva @animate
  • Debug: Stato e log ispezionabili con @state e @log
  • Compatibilità: Tutti i browser moderni
  • Licenza: MIT
  • Autore: devben

Per approfondire: consulta la documentazione su GitHub o esplora gli esempi inclusi.
