1. Quick Setup

<script src="ayisha.js"></script>
Tip: Just include the script and use Ayisha.js directives in your HTML!

2. Basic Binding with @model

<input @model="name" placeholder="Your name">
Hello {{name}}!
Hello {{name}}!
Tip: @model binds the input value to a variable. Use {{name}} to display it anywhere.

3. Events: @click, @input, @focus, @blur

<button @click="count++">Increment</button>
<input @input="msg = 'Typed!'" placeholder="Type here...">
<input @focus="msg = 'Focused!'" @blur="msg = 'Blurred!'">
Value: {{count}} | Message: {{msg}}
Value: {{count}} | Message: {{msg}}
Tip: Use event directives to react to user actions. You can update any variable in the handler.

4. Conditional Rendering: @if, @show, @hide

<button @click="show = !show">Toggle</button>
<div @if="show">Visible with @if</div>
<div @show="show">Visible with @show</div>
<div @hide="show">Hidden when show is true</div>
Visible with @if
Visible with @show
Hidden when show is true
Tip: @if adds/removes from DOM, @show toggles display, @hide is the opposite of @show.

5. Loops: @for, @key

<ul>
  <li @for="item in items" @key="item.id">{{item.name}}</li>
</ul>
  • {{item.name}}
Tip: @for loops over arrays. @key helps with efficient updates.

6. Array Operations: @map, @filter, @reduce, @source, @initial

<div @source="numbers" @map="n => n * 2" @result="doubled"></div>
<ul>
  <li @for="n in doubled">{{n}}</li>
</ul>
<div @source="numbers" @filter="n => n % 2 === 0" @result="evens"></div>
<ul>
  <li @for="n in evens">{{n}}</li>
</ul>
<div @source="numbers" @reduce="(a, n) => a + n" @initial="0" @result="sum"></div>
Sum: {{sum}}
  • {{n}}
  • {{n}}
Sum: {{sum}}
Tip: @map, @filter, and @reduce let you transform arrays reactively. @source is the input array, @result is the output variable.

7. Fetching Data: @fetch, @result

<button @fetch="'https://jsonplaceholder.typicode.com/users'" @result="users">Load users</button>
<ul>
  <li @for="user in users">{{user.name}}</li>
</ul>
  • {{user.name}}
Tip: @fetch can be used on any element. @result stores the response in a variable.

Fetch with sub-directives

<button @fetch:click="'https://jsonplaceholder.typicode.com/posts'" @result="posts">Load posts</button>
<div @fetch:hover="'https://jsonplaceholder.typicode.com/todos'" @result="todos">Hover to load todos</div>
Hover to load todos
Tip: Sub-directives like @fetch:click or @fetch:hover trigger fetch on specific events.

8. Dynamic Classes & Styles: @class, @style, @class:focus, @class:hover

<input @model="isActive" type="checkbox"> Active
<div @class="{active: isActive, inactive: !isActive}" @style="{color: isActive ? 'green' : 'red'}">Status: {{isActive ? 'Active' : 'Inactive'}}</div>
<input @class:focus="{'focus-border': true}" placeholder="Focus me!">
<div @class:hover="{'hovered': true}" style="padding:1em;border:1px solid #ccc;">Hover me!</div>
Active
Status: {{isActive ? 'Active' : 'Inactive'}}
Hover me!
Tip: @class and @style can be dynamic. Sub-directives like @class:focus or @class:hover apply classes on events.

9. Text Binding: @text and sub-directives

<div @text="greeting">Default text</div>
<div @text:hover="'Hovered!'">Hover me for text change</div>
Default text
Hover me for text change
Tip: @text sets the element's text. Sub-directives like @text:hover change text on events.

10. Switch/Case: @switch, @case, @default

<select @model="role">
  <option value="user">User</option>
  <option value="admin">Admin</option>
</select>
<div @switch="role">
  <div @case="'user'">Welcome, user!</div>
  <div @case="'admin'">Welcome, admin!</div>
  <div @default>Please select a role.</div>
</div>
role = 'norole'
Welcome, user!
Welcome, admin!
Please select a role.
Tip: @switch renders only the matching @case or @default.

11. Animations: @animate

<button @click="showAnim = !showAnim">Toggle Animation</button>
<div @if="showAnim" @animate="fade-in">Animated in!</div>
Animated in!
Tip: @animate adds a CSS animation when the element appears.

12. Set Variable: @set and sub-directives

<button @set:click="foo = 42">Set foo to 42</button>
Value: {{foo}}
Value: {{foo}}
Value: {{cane}}
Tip: @set assigns a value to a variable on an event.

13. Validation: @validate (advanced)

<input type="text" @model="zip" @validate="^\d{5}$" placeholder="ZIP code (5 digits)">
<div @show="!zip" style="color:#aaa;">Enter a ZIP code to validate</div>
<div @show="zip && _validate && _validate.zip" style="color:#2ecc40;">Valid ZIP!</div>
<div @show="zip && _validate && !_validate.zip" style="color:#ff4136;">Invalid ZIP!</div>
Enter a ZIP code to validate
Valid ZIP!
Invalid ZIP!
Tip: @validate supports custom regex and multiple rules (comma separated).

14. Components: <component> and @src

<component @src="component.html"></component>
Tip: <component> loads and renders external HTML as a reusable component.

15. State & Debug: @state, @log

<div @state></div>
<div @log></div>
Tip: @state shows the full state object. @log prints debug info for the current context.
