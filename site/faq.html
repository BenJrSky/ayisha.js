Frequently Asked Questions

Does Ayisha.js work without a build step?

Yes, just include <script src="ayisha.js"></script> and you can use all directives directly in HTML.

How do I manage state?

Use the state variable for all reactive properties. For example: state.name, state.count.

How does validation work?

With @validate on inputs and by checking _validate for results.

Can I create reusable components?

Yes, with the tag <component @src="file.html"></component>.

How do I do SPA routing?

Use @page for sections and @link for navigation buttons/links.

Can I use fetch and dynamic data?

Yes, with @fetch and @result you can load data from APIs and use it immediately.

How do I debug?

Use @log to see directive state and @state to see the global state.

Is Ayisha.js suitable for large projects?

It's designed for small/medium projects, prototypes, demos, landing pages, and lightweight apps. For enterprise projects, consider more structured frameworks.
