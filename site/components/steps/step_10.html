Step 10

🎠 Image Carousel

Build an interactive image carousel with navigation!

📝 The Code:

<div class="carousel">
  <img @src="images[currentImage]" alt="Slide">
  <button @click="currentImage = (currentImage - 1 + images.length) % images.length">⬅️</button>
  <button @click="currentImage = (currentImage + 1) % images.length">➡️</button>
</div>

<div class="dots">
  <span @map="images" @click="currentImage = index" 
        @class="{'active': index === currentImage}">●</span>
</div>

🎮 Try it:

Click arrows or dots to navigate

🎯 What you learned:
  • Modulo operator % for circular navigation
  • @src for dynamic image sources
  • Array indexing: images[currentImage]
  • CSS positioning for carousel controls
