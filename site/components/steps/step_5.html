Step 5

🔍 Conditional Display - Show and Hide Things

Learn to show/hide elements based on conditions. Very useful for dynamic interfaces!

📝 The Code:

<button @click="showSecret = !showSecret">Toggle Secret</button>
<div @if="showSecret">🔐 This is a secret message!</div>
<div @if="!showSecret">🔒 Secret is hidden</div>

<!-- Show vs If difference -->
<div @show="showSecret">📱 I stay in DOM but invisible</div>
<div @if="showSecret">🗑️ I get removed from DOM completely</div>

🎮 Try it:

showSecret = false;
🔐 Secret revealed! This element is created/destroyed with @if
🔒 Secret hidden! Click the button to reveal it
Difference between @show and @if:
📱 @show: I'm always in the DOM, just hidden when false
🗑️ @if: I'm completely removed from DOM when false
🎯 What you learned:
  • @if="condition" completely removes/adds elements
  • @show="condition" just hides/shows (stays in DOM)
  • Use ! for NOT conditions
  • Ternary operator condition ? 'yes' : 'no' for text
