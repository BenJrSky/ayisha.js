4. Conditional Rendering: Complete Guide

4.1 Basic Conditional Directives

<button @click="show = !show">Toggle</button>
<div @if="show">Visible with @if (DOM add/remove)</div>
<div @show="show">Visible with @show (display toggle)</div>
<div @hide="show">Hidden when show is true</div>
show = false;
✓ Visible with @if (DOM add/remove)
✓ Visible with @show (display toggle)
✓ Hidden when show is true (@hide directive)

Performance Comparison:

  • @if: Better for content that changes rarely - completely removes from DOM
  • @show/@hide: Better for frequent toggles - just changes CSS display property

4.2 Nested Conditional Logic

<select @model="userRole">
  <option value="">Select Role</option>
  <option value="guest">Guest</option>
  <option value="user">User</option>
  <option value="admin">Admin</option>
</select>

<div @if="userRole">
  <div @if="userRole === 'guest'">
    <p>Welcome, Guest! Limited access.</p>
  </div>
  <div @if="userRole === 'user'">
    <p>Welcome, User! Standard access.</p>
  </div>
  <div @if="userRole === 'admin'">
    <p>Welcome, Admin! Full access.</p>
  </div>
</div>

<div @if="!userRole">
  <p>Please select a role to see available options.</p>
</div>
userRole = ''; guestActions = true; showUserPanel = false; adminMode = 'basic';
