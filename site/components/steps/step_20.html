6. Functional Array Operations: Complete Guide

Transform, filter, and process arrays declaratively using functional programming patterns.

6.1. Transform Array with @map

<!-- Basic transformation -->
<div @source="numbers" @map="n => n * 2" @result="doubled"></div>
<!-- Complex object transformation -->
<div @source="products" @map="p => ({...p, totalPrice: p.price * p.quantity})" @result="enhancedProducts"></div>

6.2. Filter Array with @filter

<!-- Simple filtering -->
<div @source="numbers" @filter="n => n % 2 === 0" @result="evens"></div>
<!-- Complex object filtering with multiple conditions -->
<div @source="enhancedProducts" @filter="p => p.category === 'Electronics' && p.totalPrice > 500" @result="expensiveElectronics"></div>

6.3. Reduce Array with @reduce and @initial

<!-- Simple sum -->
<div @source="numbers" @reduce="(a, n) => a + n" @initial="0" @result="sum"></div>
<!-- Complex analytics -->
<div @source="enhancedProducts" 
     @reduce="(acc, p) => ({...acc, totalRevenue: acc.totalRevenue + p.totalPrice, itemCount: acc.itemCount + p.quantity})" 
     @initial="{totalRevenue: 0, itemCount: 0}" 
     @result="analytics"></div>

6.4. Chain Operations for Complex Data Processing

<!-- Multi-step data pipeline -->
<!-- Step 1: Filter products by category -->
<div @source="products" @filter="p => p.category === 'Electronics'" @result="electronics"></div>
<!-- Step 2: Add calculated fields -->
<div @source="electronics" @map="p => ({...p, totalValue: p.price * p.quantity, discount: p.price > 500 ? 0.1 : 0})" @result="processedElectronics"></div>
<!-- Step 3: Apply discounts -->
<div @source="processedElectronics" @map="p => ({...p, finalPrice: p.price * (1 - p.discount)})" @result="finalElectronics"></div>
<!-- Step 4: Calculate total savings -->
<div @source="finalElectronics" @reduce="(acc, p) => acc + (p.price * p.discount * p.quantity)" @initial="0" @result="totalSavings"></div>
