🌐 API Integration

Fetch data from APIs and display it dynamically!

📝 The Code:

<button @click="fetchData()">Load Jokes</button>
<div @if="loading">⏳ Loading...</div>
<div @if="error">❌ Error: {{error}}</div>
<div @if="jokes">
  <div @map="jokes">
    <p>😄 {{item.setup}}</p>
    <p><strong>{{item.punchline}}</strong></p>
  </div>
</div>

🎮 Try it:

loading = false; error = null; jokes = null; fetchData = async function() { loading = true; error = null; try { const response = await fetch('https://official-joke-api.appspot.com/jokes/programming/ten'); jokes = await response.json(); } catch (e) { error = 'Failed to load jokes'; } loading = false; };
⏳

Loading jokes...

❌ Error: {{error}}
