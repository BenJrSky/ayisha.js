3.3 Sub-directives for Specific Events

<!-- Using sub-directives for better control -->
<div @click="status='Clicked'" @hover="status='Hovered'" @leave="status='Left'">
  Multi-event element
</div>
<button @fetch:click="'https://api.github.com/users/octocat'" @result="userData">
  Fetch on Click
</button>
<input @class:focus="{'focused': true}" @class:blur="{'focused': false}" placeholder="Dynamic classes on focus/blur">
status = 'Ready';
Multi-event element (Status: {{status}})

User: {{userData.name}} (@{{userData.login}})

Bio: {{userData.bio}}

Public Repos: {{userData.public_repos}}

Sub-directive Advantage: More precise control over when actions occur. You can have different behaviors for different events on the same element.
