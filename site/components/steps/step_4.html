4. Liste e Iterazione

Impara a lavorare con liste di dati usando la direttiva @for per creare elementi dinamici.

4.1. Iterazione di Base

<!-- Lista semplice -->
<ul>
  <li @for="item in lista">{{item}}</li>
</ul>

<!-- Con indice -->
<ul>
  <li @for="i, item in lista">{{i}}: {{item}}</li>
</ul>

🎮 Prova tu:

frutti = ['Mela', 'Banana', 'Arancia', 'Fragola', 'Uva']; nuovoFrutto = ''; function aggiungiFrutto() { if (nuovoFrutto.trim()) { frutti.push(nuovoFrutto); nuovoFrutto = ''; } } function rimuoviFrutto(indice) { frutti.splice(indice, 1); }

Lista frutti:

4.2. Iterazione di Oggetti

<!-- Lista di oggetti -->
<div @for="prodotto in prodotti" @key="prodotto.id">
  <h4>{{prodotto.nome}}</h4>
  <p>Prezzo: €{{prodotto.prezzo}}</p>
</div>

4.3. Liste Annidate

<!-- Liste dentro liste -->
<div @for="categoria in categorie">
  <h3>{{categoria.nome}}</h3>
  <ul>
    <li @for="item in categoria.items">{{item}}</li>
  </ul>
</div>

🎯 Cosa hai imparato:
  • @for="item in array" - Itera su un array
  • @for="i, item in array" - Itera con indice
  • @key="item.id" - Identificatore unico per performance
  • Iterazione di oggetti complessi
  • Liste annidate e strutture dati complesse
  • Filtri e manipolazione di liste
