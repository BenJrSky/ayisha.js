Step 9

🎨 Dynamic Styling and Advanced Interactions

Learn to change styles dynamically and create complex interactions with multiple assignments!

📝 The Code:

<div @style="{'background': bgColor, 'padding': '20px'}">
  Styled dynamically!
</div>

<input @model="bgColor" type="color" value="#ff6b6b">
<input @model="textSize" type="range" min="12" max="48">

<p @style="{'font-size': textSize + 'px', 'color': bgColor}">
  Dynamic text!
</p>

<!-- Multiple assignments in one click -->
<button @click="bgColor='#9C27B0'; textSize=32; borderRadius=0">
  Purple Theme
</button>

🎮 Try it:

bgColor = '#ff6b6b'; textSize = 18; borderRadius = 8;
{{bgColor}}

🎯 What you learned:
  • @style="{'property': value}" for dynamic CSS
  • Color and range inputs work great with @model
  • Multiple assignments: @click="var1=val1; var2=val2; var3=val3"
  • String concatenation with + for units like 'px'
  • CSS transitions make changes smooth and beautiful
