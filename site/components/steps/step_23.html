9. Text and Content Manipulation

Master dynamic text content, formatting, and interactive text manipulation patterns.

9.1 Dynamic Text Content

<div @text="greeting">Default text</div>
<div @text:hover="'Hovered!'">Hover me for text change</div>
<div @text:click="'Clicked!'" @text:dblclick="'Double clicked!'">Click or double-click me</div>

🎮 Try it:

greeting = 'Hello World'; textState = 'ready';
Default text
Hover me for text change
Click or double-click me

9.2 Advanced Text Formatting and Templates

<!-- Dynamic text with complex expressions -->
<input @model="userName" placeholder="Enter your name">
<input @model="userAge" type="number" placeholder="Enter your age">

<div @text="userName && userAge ? 
           `Hello ${userName}! You are ${userAge} years old. ${
             userAge < 18 ? 'You are a minor.' : 
             userAge < 65 ? 'You are an adult.' : 
             'You are a senior.'
           }` :
           'Please enter your name and age above.'"></div>

🎮 Try it:

userName = ''; userAge = ''; formatType = 'uppercase'; inputText = 'Hello World';

9.3 Interactive Text Effects

<!-- Text that changes based on interaction -->
<div @text:hover="'Hover effect active!'" 
     @text:leave="'Hover to activate'"
     @style:hover="{fontSize: '1.2em', color: '#2196F3'}">
  Interactive text
</div>

<!-- Typing effect simulation -->
<div @text="typedText"></div>
<button @click="startTyping()">Start Typing Effect</button>

🎮 Try it:

typedText = ''; fullText = 'This is a typing effect simulation!'; currentIndex = 0; isTyping = false; startTyping = function() { if (isTyping) return; isTyping = true; typedText = ''; currentIndex = 0; typeNextChar(); }; typeNextChar = function() { if (currentIndex < fullText.length) { typedText = fullText.substring(0, currentIndex + 1); currentIndex++; setTimeout(typeNextChar, 100); } else { isTyping = false; } };
