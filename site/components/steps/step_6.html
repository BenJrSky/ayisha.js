6. Classi CSS e Stili Dinamici

Impara a applicare stili CSS dinamicamente usando le direttive @class e @style per creare interfacce reattive.

6.1. Direttiva @class

<!-- Classe dinamica basata su condizione -->
<div @class="isActive ? 'active' : 'inactive'">
  Contenuto
</div>

<!-- Più classi dinamiche -->
<button @class="status === 'success' ? 'btn-success' : 'btn-danger'">
  {{buttonText}}
</button>

🎮 Prova tu:

theme = 'light'; isActive = false; status = 'neutral'; size = 'medium'; function toggleActive() { isActive = !isActive; } function changeStatus() { const statuses = ['success', 'warning', 'danger', 'info']; const currentIndex = statuses.indexOf(status); status = statuses[(currentIndex + 1) % statuses.length]; }

6.2. Direttiva @style

<!-- Stili inline dinamici -->
<div @style="backgroundColor: themeColor; width: size + 'px'">
  Contenuto
</div>

<!-- Animazioni e transizioni -->
<div @style="transform: 'rotate(' + rotation + 'deg)'">
  Elemento rotante
</div>

🎮 Prova tu:

     @style="'animation-duration:' + duration + 's'">
  Elemento animato
</div>
