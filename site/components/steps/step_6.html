Step 6

📋 Lists and Loops - Working with Arrays

Display lists of items dynamically. Essential for any data-driven app!

📝 The Code:

<ul @map="todoList">
  <li>{{item}} <button @click="todoList.splice(index, 1)">❌</button></li>
</ul>

<input @model="newTodo" placeholder="Add new todo...">
<button @click="todoList.push(newTodo); newTodo = ''">➕ Add</button>

🎮 Try it:

todoList = ['Learn Ayisha.js', 'Build awesome app', 'Share with friends']; newTodo = '';
  • ✓ {{item}}
📊 Todo Stats:

Total items: {{todoList.length}}

List content: {{todoList.join(', ') || 'Empty list'}}

🎯 What you learned:
  • @map="arrayName" loops through arrays
  • Inside loops: item = current item, index = position
  • array.push() adds items, array.splice() removes
  • Multiple actions in one click: @click="action1; action2"
