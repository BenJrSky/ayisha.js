2. Event Handling e Interazioni

Scopri come gestire eventi e creare interazioni con l'utente usando le direttive di Ayisha.js.

2.1. Eventi Click e Modifica

<!-- Gestione click -->
<button @click="contatore++">Incrementa</button>
<button @click="reset()">Reset</button>

<!-- Eventi di input -->
<input @input="aggiorna()" @model="testo">

🎮 Prova tu:

contatore = 0; testo = ''; ultimoClick = null; function reset() { contatore = 0; testo = ''; ultimoClick = 'Reset eseguito!'; } function aggiorna() { ultimoClick = 'Testo modificato: ' + testo; }

Contatore:

Testo interattivo:

Contenuto: {{testo || 'Nessun testo'}}

Ultimo evento: {{ultimoClick || 'Nessun evento'}}

2.2. Eventi di Tastiera

<!-- Gestione tastiera -->
<input @keydown="gestisciTastiera($event)" @model="input">
<input @keydown.enter="conferma()" @model="messaggio">

🎮 Prova tu:

input = ''; messaggio = ''; tastiPremuti = []; messaggiConfermati = []; function gestisciTastiera(event) { const tasto = event.key; tastiPremuti.push(tasto); if (tastiPremuti.length > 10) { tastiPremuti.shift(); } } function conferma() { if (messaggio.trim()) { messaggiConfermati.push(messaggio); messaggio = ''; } }

Monitor tastiera:

Tasti premuti: {{tastiPremuti.join(', ') || 'Nessuno'}}

Conferma con Enter:

Messaggi confermati:
🎯 Cosa hai imparato:
  • @click="espressione" - Gestione eventi click
  • @input="funzione()" - Evento quando l'input cambia
  • @keydown="funzione($event)" - Gestione eventi tastiera
  • @keydown.enter="funzione()" - Evento specifico per tasto Enter
  • $event - Accesso all'oggetto evento
  • Funzioni personalizzate nel blocco <init>
