7. Watchers e Proprietà Computate

Impara a reagire ai cambiamenti di dati e creare valori derivati automaticamente con @watch e funzioni computate.

7.1. Direttiva @watch

<!-- Reagisce ai cambiamenti di una variabile -->
<div @watch="nome">
  <p>Nome è cambiato: {{nome}}</p>
</div>

<!-- Watch multipli -->
<div @watch="nome, cognome">
  <p>Nome completo: {{nome}} {{cognome}}</p>
</div>

function getNomeCompleto() {
  return nome + ' ' + cognome;

function getStatisticheCarrello() {
  return {
    totale: carrello.reduce((sum, item) => sum + item.prezzo, 0),
    numeroArticoli: carrello.length
  };
}
