Step 7

📝 Forms and Validation

Build proper forms with validation. Essential for user input!

📝 The Code:

<form @submit="submitForm()">
  <input @model="email" @input="validateEmail()" placeholder="Email">
  <div @if="emailError" style="color:red">{{emailError}}</div>
  
  <input @model="password" type="password" placeholder="Password">
  <div @if="password && password.length < 6" style="color:red">
    Password too short!
  </div>
  
  <button type="submit" @disabled="emailError || !email || !password">
    Submit
  </button>
</form>

🎮 Try it:

email = ''; password = ''; emailError = ''; formMessage = ''; validateEmail = function() { if (email && !email.includes('@')) { emailError = 'Please enter a valid email'; } else { emailError = ''; } }; submitForm = function() { if (!emailError && email && password) { formMessage = '✅ Form submitted successfully!'; } };
{{emailError}}
Password must be at least 6 characters!
{{formMessage}}
🎯 What you learned:
  • @input="function()" runs code when typing
  • Combine conditions with && and ||
  • Functions can be stored in variables and called
  • Dynamic styling with @style for visual feedback
