Step 3

🔘 Buttons and Actions

Now let's add some buttons to perform actions. We'll create a simple counter.

📝 The Code:

<button @click="count++">➕ Add 1</button>
<button @click="count--">➖ Subtract 1</button>
<button @click="count = 0">🔄 Reset</button>
<p>Count: {{count || 0}}</p>

🎮 Try it:

count = 0;

Count: {{count}}

🎯 What you learned:
  • @click="expression" runs code when clicked
  • count++ increases the number
  • count = 0 sets a specific value
  • Variables update the display automatically
