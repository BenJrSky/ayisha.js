Step 3

3. Rendering Condizionale

Impara a mostrare o nascondere elementi basandoti sulle condizioni con le direttive @if, @show e @switch.

3.1. Direttiva @if

<!-- Mostra solo se la condizione è vera -->
<div @if="utente.isLogged">
  <p>Benvenuto, {{utente.nome}}!</p>
</div>

<!-- Condizioni multiple -->
<div @if="eta >= 18">Sei maggiorenne</div>
<div @if="eta < 18">Sei minorenne</div>

🎮 Prova tu:

utente = { nome: 'Mario', isLogged: false, tipo: 'ospite' }; eta = 25; mostraDettagli = false;

Stato utente:

🎉 Benvenuto, {{utente.nome}}!

Hai accesso completo al sistema.

⚠️ Effettua il login per accedere.

Controllo età:

✅ Sei maggiorenne ({{eta}} anni)
❌ Sei minorenne ({{eta}} anni)
🎂 Hai diritto agli sconti senior!

3.2. Direttiva @show

<!-- Usa display:none invece di rimuovere l'elemento -->
<div @show="mostraMessaggio">
  <p>Questo messaggio viene nascosto con display:none</p>
</div>

🎮 Prova tu:

mostraMessaggio = true; animazione = false;

Differenza tra @if e @show:

Con @if (elemento rimosso dal DOM):
🎯 Questo elemento viene creato/distrutto ogni volta
Con @show (elemento nascosto con display:none):
🎯 Questo elemento è sempre nel DOM, solo nascosto

3.3. Direttiva @switch

<!-- Switch per gestire più condizioni -->
<div @switch="tipoUtente">
  <div @case="admin">Pannello Amministratore</div>
  <div @case="utente">Pannello Utente</div>
  <div @default>Accesso Ospite</div>
</div>

🎮 Prova tu:

tipoUtente = 'ospite'; tema = 'chiaro';

Selezione tipo utente:

🚪 Accesso Ospite

Benvenuto! Hai accesso limitato al sistema.

  • Puoi leggere i contenuti pubblici
  • Non puoi commentare o modificare

👤 Pannello Utente

Benvenuto nel tuo pannello personale!

  • Puoi leggere tutti i contenuti
  • Puoi commentare e interagire
  • Puoi modificare il tuo profilo

⚙️ Pannello Amministratore

Benvenuto nel pannello di controllo!

  • Controllo completo del sistema
  • Gestione utenti e contenuti
  • Accesso alle statistiche

🛡️ Pannello Moderatore

Benvenuto nel pannello di moderazione!

  • Puoi moderare i contenuti
  • Puoi gestire i commenti
  • Puoi bloccare utenti problematici

❓ Tipo utente sconosciuto

Contatta l'amministratore per risolvere il problema.

🎯 Cosa hai imparato:
  • @if="condizione" - Mostra/nasconde rimuovendo dal DOM
  • @show="condizione" - Mostra/nasconde con display:none
  • @switch="variabile" - Gestisce più condizioni
  • @case="valore" - Caso specifico in uno switch
  • @default - Caso di default quando nessun case corrisponde
  • Differenza tra @if e @show per performance
