7. API Integration: @fetch and Data Loading

Learn to fetch data from APIs and handle loading states, errors, and different trigger events.

7.1 Basic Fetch Operations

<button @fetch="'https://jsonplaceholder.typicode.com/users'" @result="users">Load Users</button>
<div @if="users">
  <h4>Users ({{users.length}})</h4>
  <ul>
    <li @for="user in users" @key="user.id">{{user.name}} - {{user.email}}</li>
  </ul>
</div>

7.2 Event-Triggered Fetch with Sub-directives

<!-- Different trigger events -->
<button @fetch:click="'https://jsonplaceholder.typicode.com/posts'" @result="posts">Load Posts on Click</button>
<div @fetch:hover="'https://jsonplaceholder.typicode.com/todos'" @result="todos">Hover to Load Todos</div>
<input @fetch:focus="'https://jsonplaceholder.typicode.com/comments'" @result="comments" placeholder="Focus to load comments">

7.3 Real-World Example: Dynamic Search

<!-- Search with debounced input -->
<input @model="searchTerm" 
       @input="searchTerm.length > 2 ? (searchUsers = true) : (searchUsers = false)"
       placeholder="Search users (type at least 3 characters)">
       
<div @if="searchUsers && searchTerm.length > 2"
     @fetch="'https://jsonplaceholder.typicode.com/users'"
     @result="allUsers"></div>
     
<div @if="allUsers"
     @source="allUsers"
     @filter="user => user.name.toLowerCase().includes(searchTerm.toLowerCase()) || user.email.toLowerCase().includes(searchTerm.toLowerCase())"
     @result="filteredUsers"></div>
