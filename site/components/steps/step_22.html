8. Dynamic Styling: @class and @style Mastery

Create rich, interactive UIs with dynamic classes and styles that respond to data and user interactions.

8.1 Basic Dynamic Classes and Styles

<input @model="isActive" type="checkbox"> Active
<div @class="{active: isActive, inactive: !isActive}" @style="{color: isActive ? 'green' : 'red'}">
  Status: {{isActive ? 'Active' : 'Inactive'}}
</div>

🎮 Try it:

isActive = false;
Status: {{isActive ? 'Active' : 'Inactive'}}

8.2 Event-Based Styling with Sub-directives

<!-- Focus and hover styling -->
<input @class:focus="{'focus-border': true}" @class:blur="{'focus-border': false}" placeholder="Focus me!">
<div @class:hover="{'hovered': true}" @style:hover="{transform: 'scale(1.05)'}">Hover me!</div>

<!-- Interactive button with multiple states -->
<button @class:hover="{'btn-hover': true}" 
        @class:active="{'btn-active': true}"
        @style:hover="{background: '#2196F3', transform: 'translateY(-2px)'}"
        @style:active="{transform: 'translateY(0px)'}">
  Interactive Button
</button>

🖱️ Hover me for smooth transformation!

8.3 Complex Conditional Styling

<!-- Status indicator with multiple states -->
<select @model="status">
  <option value="success">Success</option>
  <option value="warning">Warning</option>
  <option value="error">Error</option>
  <option value="info">Info</option>
</select>

<div @class="{
       'status-success': status === 'success',
       'status-warning': status === 'warning', 
       'status-error': status === 'error',
       'status-info': status === 'info'
     }"
     @style="{
       background: status === 'success' ? '#e8f5e9' : status === 'warning' ? '#fff3e0' : status === 'error' ? '#ffebee' : '#e3f2fd',
       color: status === 'success' ? '#2e7d32' : status === 'warning' ? '#ef6c00' : status === 'error' ? '#c62828' : '#1565c0',
       border: '2px solid ' + (status === 'success' ? '#4caf50' : status === 'warning' ? '#ff9800' : status === 'error' ? '#f44336' : '#2196f3')
     }">
  Status: {{status || 'Select a status'}}
</div>

🎮 Try it:

status = 'info'; progressValue = 50; taskPriority = 'medium';
{{status === 'success' ? '✅' : status === 'warning' ? '⚠️' : status === 'error' ? '❌' : 'ℹ️'}} Status: {{status ? status.toUpperCase() : 'Select a status'}}
{{progressValue}}%

{{taskPriority === 'urgent' ? '🚨' : taskPriority === 'high' ? '🔥' : taskPriority === 'medium' ? '📋' : '📝'}} {{taskPriority.toUpperCase()}} Priority Task

This task has {{taskPriority}} priority level and requires appropriate attention.
