Caratteristiche Principali

Ayisha.js offre tutto ciò di cui hai bisogno per creare interfacce moderne e reattive senza dipendenze esterne e con una curva di apprendimento dolce.

⚡

Reattività

Ogni cambiamento nello stato aggiorna automaticamente l'interfaccia. Niente più DOM manipulation manuale!

🎯

Direttive

Usa attributi speciali direttamente nell'HTML: @if, @for, @click, @model, @fetch e molte altre.

🧩

Componenti

Includi parti di pagina da file esterni o creale inline per un codice modulare e riutilizzabile.

🔗

Data Binding

Collega facilmente i dati agli input e viceversa con binding bidirezionale automatico.

🛣️

Routing SPA

Gestisce pagine diverse senza ricaricare il sito, perfetto per Single Page Applications.

🔍

Debug Tools

Sistema integrato di logging ed errori per un debugging rapido ed efficace.

Direttive Disponibili

Un set completo di direttive per ogni esigenza di sviluppo web moderno.

🎛️ Controllo di Flusso

@if, @show, @hide, @for, @switch, @case, @default

📊 Data Binding

@model, @text, @class, @style

🖱️ Eventi

@click, @hover, @focus, @blur, @change, @input

🌐 HTTP

@fetch, @result, @method, @headers, @payload

👁️ Osservatori

@watch per osservare cambiamenti di stato

🧩 Componenti

@component, @src per modularità

🛣️ Navigazione

@link, @page per SPA routing

🔧 Utilità

@set, @validate (con regex, email, phone, min/max), @animate, @state, @log

⚙️ Funzionale

@source, @map, @filter, @reduce

2. Todo List con @for e @model

<div class="todo-demo"> <input @model="newTodo" placeholder="Nuovo todo..."> <button class="btn-demo" @click="newTodo.trim() && todos.push({id: Date.now(), text: newTodo, done: false}); newTodo = ''"> Aggiungi </button> <div @for="todo in todos" class="todo-item" @class="{completed: todo.done}"> <span>{{todo.text}}</span> <button class="btn-demo" @click="todo.done = !todo.done"> {{todo.done ? '↶' : '✓'}} </button> </div> </div> <init> state.todos = []; state.newTodo = ''; </init>
{{todo.text}}

4. Rendering Condizionale con @if/@show/@hide
